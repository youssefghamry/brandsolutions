Map Configuration [SS]<\/strong><\/h2>\n\n\n\n

Client should buy Map API from Google for enabling the maps into the panels. Without buying those APIs clients cannot load Google maps into the panels for selecting zones. For generating map api key you can watch this video. Now go to your admin panel then \u201cThird party APIs” menu, here you will find two inputs for map api key client and map api key server. You can restrict the client with admin panel domain and the server key with your server ip address. If you don\u2019t want any restriction then you can use single api key for both field.<\/p>\n\n\n\n
